# Frontend Routes

| Route | Description |

| `/about` | About page - Information about hanachan.org |
| `/admin` | Admin panel - Administrative dashboard |
| `/chat` | AI Chat - Hanachan AI tutor conversations |
| `/dashboard` | **Main Dashboard** - Progress, achievements, weekly goals |
| `/dictionary` | Dictionary - Kanji and vocabulary lookup engine |
| `/downloads` | Downloads - Downloadable resources and materials |
| `/flashcards` | Flashcards - SRS flashcard study system |
| `/game` | Games Hub - Unified portal for interactive learning games |
| `/grammarpoint` | Grammar detail - Individual grammar point pages |
| `/kanji-dashboard` | Kanji overview - Kanji learning progress |
| `/knowledge-base` | Knowledge Base - Kanji, grammar, verbs, kana reference |
| `/landing` | Alternative landing page |
| `/library` | Library - Podcasts, reading, and mnemonics |
| `/library/podcasts` | Podcasts & Videos - Curated + personal YouTube videos |
| `/library/reading` | Japanese Reading - Short stories with translations |
| `/library/mnemonics` | Mnemonics - Memory aids for Kanji |
| `/login` | Login page - User authentication |
| `/mp3` | Audio files - MP3 audio resources |
| `/practice` | Practice Hub - Structured training, quizzes, and JLPT protocols |
| `/privacy-policy` | Privacy Policy - Legal privacy information |
| `/quoot` | Quoot - High-speed vocabulary battle game |
| `/study-plan` | Study Plan - JLPT study plan creation & management |
| `/terms-of-service` | Terms of Service - Legal terms |
| `/tools` | Tools - Text parser, translator, quick lookup tools |
| `/vocabulary-map` | Vocabulary Map - Visual vocabulary explorer |
